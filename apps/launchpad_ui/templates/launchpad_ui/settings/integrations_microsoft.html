Setup Guide

Create an Entra app registration for SnipeOps, then grant Microsoft Graph application permissions.

  1. Open the Microsoft Entra admin center.
  2. Create or select an App Registration for Launchpad / SnipeOps.
  3. Create a client secret and paste it above.
  4. Add Microsoft Graph application permissions for Intune device reads.
  5. Grant admin consent.
