# Launchpad Admin Guide

Launchpad is an internal operations platform for district technology workflows. This admin guide documents setup, configuration, day-to-day administration, and troubleshooting for Launchpad and its built-in applications.

## Who this guide is for

This guide is intended for Launchpad administrators, technology managers, and staff who maintain the application, configure integrations, manage users, or support Launchpad applications.

## Recommended setup order

1. Complete the local administrator setup.
2. Configure General Settings, especially the public base URL and timezone.
3. Configure Email Integration and send a test message.
4. Configure Authentication and access rules.
5. Create or import users and assign groups/permissions.
6. Configure application-specific settings such as Staff Status, Finance, and SnipeOps.
